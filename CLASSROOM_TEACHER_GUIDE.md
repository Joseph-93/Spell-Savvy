## ğŸ’¡ Tips & Best Practices

### Naming Classrooms
**Good names:**
- "Period 1", "Period 2", etc.
- "Monday 9am", "Tuesday 2pm"
- "Advanced Spelling", "Beginner Spelling"
- "Class A", "Class B"

**Less helpful:**
- "My Class" (if you have multiple)
- "Students" (too generic)

### Multiple Periods?
Create separate classrooms for each:
- Period 1
- Period 2
- Period 3
- etc.

Each gets its own join code!

### Reusing Next Year?
Two options:

**Option 1: Delete old, create new**
- Delete last year's classroom
- Create "Class of 2026"
- Get fresh join code

**Option 2: Keep for records**
- Create new classroom
- Leave old one (students remain for reference)

### Security
- Don't post join links publicly online
- Regenerate codes if needed
- Delete inactive classrooms

## â“ Common Questions

**Q: Can students be in multiple classrooms?**
A: Currently no. Each student is in one classroom.

**Q: What if I teach the same students different subjects?**
A: Create one classroom per subject (e.g., "English 101", "Spelling 101")

**Q: Can students join without a code?**
A: Yes, they can still manually select you as their teacher (legacy method)

**Q: How many classrooms can I create?**
A: Unlimited!

**Q: Can I move a student between classrooms?**
A: Currently requires admin access. Contact support.

**Q: What happens to students if I delete a classroom?**
A: They're unassigned from the classroom but their accounts and progress remain.

**Q: Can other teachers see my join codes?**
A: No, only you can see your classroom join codes.

**Q: Do join codes expire?**
A: No, they work indefinitely until you regenerate or delete the classroom.

**Q: Case sensitive?**
A: No, join codes work regardless of capitalization.
